# Sorting Algorithms (C++)

This folder contains my implementations of fundamental **Sorting Algorithms** in C++ as part of my Data Structures and Algorithms practice.

## 📌 Algorithms Included

### 1️⃣ Bubble Sort
- Repeatedly compares adjacent elements
- Swaps them if they are in wrong order
- Time Complexity:
  - Worst Case: O(n²)
  - Best Case: O(n)

---

### 2️⃣ Selection Sort
- Selects the smallest element
- Places it at the correct position
- Time Complexity:
  - Worst Case: O(n²)
  - Best Case: O(n²)

---

### 3️⃣ Merge Sort
- Divide and Conquer algorithm
- Recursively divides the array into halves
- Merges sorted halves
- Time Complexity:
  - Worst Case: O(n log n)
  - Best Case: O(n log n)

---

### 4️⃣ Quick Sort
- Divide and Conquer algorithm
- Selects a pivot element
- Partitions array around pivot
- Time Complexity:
  - Worst Case: O(n²)
  - Average Case: O(n log n)

---
